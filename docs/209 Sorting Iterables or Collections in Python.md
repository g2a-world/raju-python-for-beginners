* Sort below list of tuples based on **sale amount**.
```python
sales = [
    (1, 101, 500.00, 5),
    (2, 102, 250.00, 3),
    (3, 103, 1000.00, 8),
    (4, 104, 750.00, None),
    (5, 101, 300.00, -1),
    (6, 106, 400.00, 3),
    (7, 104, 200.00, 0),
    (8, 103, 150.00, 1),
    (9, 104, 600.00, 4),
    (10, 101, 800.00, 6)
]

sorted(sales, key=lambda i: i[2])

from pprint import pprint
pprint(sorted(sales, key=lambda i: i[2]))

pprint(sorted(sales, key=lambda i: i[2], reverse=True))
```
